How it works: When the AI button is clicked, contextExtractor.js traverses up the DOM from the input field to find the nearest post container. Each platform has its own post container selectors — LinkedIn uses specific article/section wrappers, X uses tweet-specific data attributes, Reddit uses post/comment thing containers.

"Change selection" flow

Clicking "Change selection" collapses the popover into a picker mode: the user clicks any post or highlights any text on the page to set it as context. The popover re-opens with the new context preview. This handles edge cases like feeds with many posts stacked together.
