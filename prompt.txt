SOCIAL AI COPILOT

---

PRODUCT VISION

Build a browser-native AI copilot that lives inside social media platforms and assists users in writing high-quality content directly in the UI.

Supported platforms:

* LinkedIn
* Facebook
* X (Twitter)
* Reddit

The assistant should feel native, fast, and invisible until needed.

---

CORE FUNCTIONALITY

When the user is on any supported platform:

1. Detect:

* Comment boxes
* Post creation inputs
* Editable fields

2. Inject:

* A minimal, premium UI trigger (for example a small "AI" button)

3. On interaction:
   Open a compact AI panel with options:

* Generate Comment
* Rewrite Selection
* Generate Post
* Quote Post (X only)

4. AI Output:

* Insert directly into active input field
* Never auto-submit

PROMPT ENGINE

System prompt:
"You are a human-like social media writer. Avoid robotic tone. Write naturally."

Dynamic inputs:

* Platform
* Task
* Tone
* Context

Platform personalities:

LinkedIn:

* Insightful, professional, not corporate

Facebook:

* Conversational, natural

X:

* Short, sharp, punchy

Reddit:

* Honest, slightly informal, value-driven, not salesy

Task types:

* Comment: add perspective, not summary
* Rewrite: improve clarity without changing meaning
* Post: structured and engaging
* Quote: opinionated and short

Rules for output:

* No robotic phrasing
* No hashtags unless explicitly requested
* No emojis unless casual tone
* Slightly imperfect human tone
